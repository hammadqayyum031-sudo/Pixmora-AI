# User Guide — PIXMORA AI

## Removing Backgrounds
1. Log in or use free browser mode
2. Upload or drag-and-drop your image
3. Select engine: IMG.LY (Free) or Remove.bg (Premium)
4. Click Process
5. Download the transparent PNG result
