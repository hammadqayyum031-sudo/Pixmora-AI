# Admin Guide — PIXMORA AI

## Access
Visit `/admin/login.php` and log in with your admin account.

## Dashboard
View users, images processed, API keys, and system status.

## Users
View, block, unblock, delete, or reset user usage.

## API Keys
Add and manage API keys (e.g., Remove.bg).

## Settings
Change site name, logo, theme, registration, upload limits, and maintenance mode.

## License
Create, activate, and manage purchase codes with domain locking.
