# PIXMORA AI — Smart AI Background Removal in Seconds

Production-ready PHP SaaS application for CodeCanyon.

## Features
- Full frontend UI (landing + dashboard + auth)
- Backend SaaS system (SQLite/MySQL)
- Admin control panel
- One-click installer
- License system with domain locking
- Secure production config loader
- Complete documentation

## Quick Start
1. Upload files to your server (PHP 7.4+)
2. Ensure `data/` and `uploads/` are writable
3. Visit `https://your-domain/install/`
4. Follow the installer wizard

## Security
- Use HTTPS in production
- config.local.php is created securely (chmod 0600)
- Input sanitization and CSRF protection
- Session security hardening
